Error Codes
===========

Introduction
------------

This section explains the meanings of error codes returned by CTS APIs.

Example of Returned Error Information
-------------------------------------

{

"error_code": "CTS.0003",

"error_msg": "Invalid message body. The message body is empty or invalid."

}
